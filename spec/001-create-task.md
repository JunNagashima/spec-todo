## 目的（何が嬉しい？）

- タスクを以下の項目で登録できる
  - タスク名
  - 登録日
  - 期限
  - 優先順位
  - ステータス

## 画面/操作（ユーザーが何をする？）

- タスク作成ボタンからタスク作成モーダルが表示
- 以下を入力できる
  - タスク名
  - 期限
  - 優先順位
- 登録日は、createdAt は自動設定
- ステータスは、登録時は TODO で固定
- 入力して登録ボタンでタスクが登録される
- 登録後にモーダルがとじる

## 入力（項目・型・必須・制約）

- タスク名
  - string
  - 必須
  - 30 文字以内
- 期限
  - date
  - 必須
  - 今日以降の日付
- 優先順位
  - HIGH | MIDDLE | LOW
  - 必須

## 受け入れ条件（Given/When/Then で 3〜5 本）

### 正常系

#### 登録モーダル開く

Given タスク一覧画面が表示されている
When 「タスクを登録する」ボタンを押す
Then タスク登録フォームモーダルが開く

#### 登録モーダル閉じる

Given タスク登録フォームモーダルが表示されている
When 「x」ボタンまたは、「モーダル枠外」を押す
Then タスク登録フォームモーダルが閉じる

#### タスク登録成功

Given タスク名「勉強する」、期限「2026-03-31」、優先順位「HIGH」が入力されている
When 登録ボタンを押す
Then 処理中は、インジケーターが表示される
And 登録処理中は登録ボタンが非活性（再押下不可）
And API が 201 を返す
And インジケーターが非表示になる
And モーダルが閉じる
And 「登録完了しました」のトーストが表示される
And トースト表示 5 秒後に非表示になる

### 異常系

#### タスク名未入力

Given タスク名が未入力
And 期限「2026-03-31」が入力されている
And 優先順位「HIGH」が選択されている
When 入力フィールドからフォーカスが外れる
Then タスク名の入力欄の下に「タスク名を入力してください。」が表示される
And 登録ボタンは非活性になる

#### 期限未入力

Given タスク名「勉強する」が入力されている
And 期限が未入力
And 優先順位「HIGH」が選択されている
When 入力フィールドからフォーカスが外れる
Then 期限の入力欄の下に「期限を入力してください。」が表示される
And 登録ボタンは非活性になる

#### 優先順位未入力

Given タスク名「勉強する」が入力されている
And 期限「2026-03-31」が入力されている
And 優先順位が未選択
When 入力フィールドからフォーカスが外れる
Then 優先順位の入力欄の下に「優先順位を入力してください。」が表示される
And 登録ボタンは非活性になる

#### 登録処理で失敗

Given タスク名「勉強する」が入力されている
And 期限「2026-03-31」が入力されている
And 優先順位「HIGH」が選択されている
When 登録ボタンを押す
Then 処理中は、インジケーターが表示される
And API が 201 などの成功以外のステータスコードを返す
And インジケーターが非表示になる
And 「登録に失敗しました」のトーストが表示される
And トースト表示 5 秒後に非表示になる
And モーダルは閉じない

## データ（更新されるテーブル/モデル、整合性ルール）

### 共通

- timezone: Asia/Tokyo
- 主キー: id(autoincrement)
- dueDate は date（日付のみ）とし、判定は Asia/Tokyo の日付（00:00 を基準）で比較する。

### tasks(新規)

※ status は、このスペックでは使用しないが TODO 以外も用意しておく

| column    | type                      | required | default       | rule           |
| --------- | ------------------------- | -------: | ------------- | -------------- |
| id        | number                    |        ○ | autoincrement | PK             |
| title     | string                    |        ○ | -             | 30 文字以内    |
| dueDate   | date （日付のみ）         |        ○ | -             | JST の今日以降 |
| priority  | enum(HIGH , MIDDLE , LOW) |        ○ | -             | -              |
| status    | enum(TODO , DOING , DONE) |        ○ | TODO          | -              |
| createdAt | timestamp                 |        ○ | now           | JST            |
| updatedAt | timestamp                 |        ○ | now           | 更新時に更新   |
