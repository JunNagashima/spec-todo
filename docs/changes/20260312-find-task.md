# タスク詳細モーダル実装

## 目的

タスクカードをクリックするとサーバーから詳細を取得し、モーダルで表示できるようにする。

## 変更点

- `src/actions/task.ts`: `getTaskById(id)` Server Action を追加
- `src/components/task/task-detail-dialog.tsx`: タスク詳細モーダルコンポーネントを新規作成
- `src/components/task/task-card.tsx`: `"use client"` 化し、`TaskDetailDialog` でラップ

## 受け入れ条件

- 初期表示時、詳細モーダルが非表示
- タスクカードクリックでモーダルが表示され、取得中はローディングスピナーが表示される
- 取得成功時: タイトル / 期限 (YYYY/MM/DD) / 優先度 (高/中/低) / ステータス / 登録日 が表示される
- X ボタンまたはモーダル枠外クリックでモーダルが閉じる
- 取得失敗時: 「取得に失敗しました」トーストが 5 秒表示され、モーダルが閉じる

## 動作確認手順

1. `bun dev` でサーバー起動
2. タスク一覧画面を開き、詳細モーダルが非表示であることを確認
3. 任意のタスクカードをクリックし、スピナー表示 → 詳細情報表示を確認
4. X ボタン / 枠外クリックでモーダルが閉じることを確認
