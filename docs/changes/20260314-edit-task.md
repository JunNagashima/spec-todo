# 20260314-edit-task

## 目的

タスク詳細モーダルに編集モードを追加する。編集モードでは保存を行わず、フォーム表示とテキスト表示を切り替えるだけ。

## 変更点

### `src/components/task/task-detail-dialog.tsx`

- `isEditing` state を追加（編集モード切替用）
- 「編集」ボタンを追加（読み取り表示時に表示）
- 編集フォームを追加（`isEditing = true` 時に表示）
  - title: `Input`（初期値 = task.title）
  - dueDate: `<input type="date">`（初期値 = task.dueDate をJST YYYY-MM-DD形式で変換）
  - priority: `Select`（HIGH/MIDDLE/LOW）
  - status: `Select`（TODO/DOING/DONE）
  - createdAt: テキスト表示のまま（編集不可）
- 「キャンセル」「更新」ボタン：どちらも `isEditing = false` に戻すだけ（保存なし）
- モーダルclose時（`handleOpenChange`）に `isEditing = false` をリセット
- `statusLabel` マッピングを追加し、ステータスの日本語表示に対応

## 受け入れ条件

- 詳細モーダルで「編集」ボタンが表示される
- 押下するとフォームに切り替わり、各フィールドに現在の値が初期値として入っている
- 「キャンセル」押下 → テキスト表示に戻り、元の値が表示される
- 「更新」押下 → テキスト表示に戻り、元の値が表示される（保存されない）
- モーダルを閉じて再度開く → 元の値が表示される

## 動作確認手順

1. `bun run dev` で開発サーバー起動
2. タスクカードクリック → 詳細モーダル表示
3. 「編集」ボタン押下 → フォーム表示に切替、各フィールドに初期値が入っていることを確認
4. 値を変更 → 「キャンセル」 → テキスト表示に戻り、元の値が表示されること
5. 値を変更 → 「更新」 → テキスト表示に戻り、元の値が表示されること
6. 値を変更 → モーダル閉じる → 再度開く → 元の値が表示されること
